# PM's Guide to Karpathy's Autoresearch

- **Author:** Aakash Gupta
- **Published:** 2026-03-20
- **URL:** https://www.news.aakashg.com/p/autoresearch-guide-for-pms

## Summary

Product management perspective on autoresearch. Key points:

- **The pattern works on anything you can score** — not just ML. PMs can use it on prompts, skills, templates.
- Author ran it on a landing page skill: 41% → 92% in 4 rounds.
- **Three requirements**: clear numeric metric, automated measurement, one file the agent can change.
- **"The human programs the organization; the agent programs the model."**
- Shopify's Lutke: 53% faster Liquid rendering, 61% fewer memory allocations from 93 automated commits.
- Covers 6 high-value PM use cases, eval templates, and a results analyzer toolkit.
- Framing: "You define what 'better' means. The agent runs the 50 rounds you'd never have time for."
