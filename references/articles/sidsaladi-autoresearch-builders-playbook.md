- **Author:** Sid Saladi
- **Published:** 2026-03-21

## Summary

Step-by-step playbook for applying autoresearch beyond ML — to skills, prompts, templates, and agent workflows. Key points:

- **No GPU or ML knowledge needed** — works with any coding agent (Claude Code, Cursor, Windsurf, Codex).
- **Cost**: ~$0.10/cycle, so 50 rounds overnight ≈ $5, 100 rounds ≈ $10-25.
- **5 optimization categories** ranked by speed of results:
  1. Claude Skills (SKILL.md files) — "the killer app for autoresearch"
  2. System prompts
  3. Content templates
  4. Agent workflows
  5. Eval criteria themselves (meta-optimization)

### Critical Lesson: Eval Criteria Are Everything
- Every criterion **must be binary** (yes/no). Sliding scales (1-7) cause the agent to game scores.
- 3-6 criteria is the sweet spot. Below 3 = loopholes. Above 6 = checklist gaming.
- "If you can't explain how to score it in one sentence, rewrite it."
- Provides copy-paste eval templates for skills, system prompts, content, and agents.

### The Three Requirements
1. Score output with a number, not a feeling
2. Scoring runs without a human
3. Only one file changes per round
